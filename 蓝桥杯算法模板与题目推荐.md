### 2️⃣ 排序 + 自定义排序

#### 模板
```cpp
struct Node {
    int x, y;
    bool operator<(const Node& other) const {
        return x < other.x;
    }
};

bool cmp(Node a, Node b) {
    return a.y < b.y;
}

sort(a.begin(), a.end(), cmp);
```

### 3️⃣ 搜索（DFS / BFS）

#### DFS 模板
```cpp
void dfs(int u) {
    visited[u] = true;
    for (auto v : graph[u])
        if (!visited[v]) dfs(v);
}
```

#### BFS 模板
```cpp
queue<int> q;
q.push(start);
while (!q.empty()) {
    int cur = q.front(); q.pop();
    for (auto next : graph[cur])
        if (!visited[next]) {
            visited[next] = true;
            q.push(next);
        }
}
```

### 5️⃣ 动态规划（DP）

#### 0/1 背包模板
```cpp
for (int i = 1; i <= n; ++i)
    for (int j = V; j >= w[i]; --j)
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
```

#### 完全背包模板
```cpp
for (int i = 1; i <= n; ++i)
    for (int j = w[i]; j <= V; ++j)
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
```

### 8️⃣ 并查集（Union-Find）

#### 模板
```cpp
int find(int x) {
    return x == parent[x] ? x : parent[x] = find(parent[x]);
}

void union_set(int a, int b) {
    parent[find(a)] = find(b);
}
```

### 9️⃣ 数学 + 数论

#### 快速幂模板
```cpp
int qmi(int a, int b, int mod) {
    int res = 1;
    while (b) {
        if (b & 1) res = (long long)res * a % mod;
        a = (long long)a * a % mod;
        b >>= 1;
    }
    return res;
}
```
